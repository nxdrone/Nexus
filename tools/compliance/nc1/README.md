# NC1 Compliance Tooling

This area is reserved for NC1-specific compliance harnesses and fixtures.

## Intended role

- Execute NC1 compliance checks against implementations and/or simulation fixtures.
- Produce repeatable evidence of alignment to NC1 contract artifacts.
- Report required, optional, and non-compliant behaviors.

## Boundary

Compliance tooling compares implementation behavior against `nexus` contract declarations.
It does **not** promote observed implementation quirks into official contract.
